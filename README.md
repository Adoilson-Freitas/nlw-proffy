- Node.js

- Knex
- React.js
- Typescript
- Styled-Components
- Axios

## Como executar 

###### git clone https://github.com/Adoilson-Freitas/nlw-proffy

### Executar Api

######  Ir para pasta backend
$ cd nlw-proffy/backend

###### Instalar dependencias
$ yarn

###### Executar aplicação
$ yarn start

Access Api http://localhost:3333/


### Executar projeto da Web

######  Ir para pasta Web
$ cd nlw-proffy/Web

###### Instalar dependencias
$ yarn

###### Executar aplicação
$ yarn start

Acessar http://localhost:3000/
